What is XT OS ?

XT OS is an Arch based post-installation script that will deploy a suite of configurations as well as associated programs for a general desktop experience.

Productivity

Choose between a wide selection of Window Managers including but not limited to AwesomeWM, DWM, i3WM, QTile and even XMonad.

Light Weight

We make sure to include a minimal install option and even software alternatives in order to limit bloat on computers.

Tailored for each user.

The post install script allows for a wide selection of pre-built configurations that allows for a quick and easy install depending on the "intention" of the build. Either for gaming, programing or even for touch screen devices.

Your Choice

We make sure our software is available to be deployed on a wide range of operating systems. Even proprietary ones.
